# 管道符

这里的管道符和liux中的管道符是一样的，都是将前面一个命令的输出，作为一个文件的形式传输给第二个命令的输入。简单的来讲第二个命令得到的是一个文件输入。

```powershell

PS C:\Users\wujiashuai> ls | format-table LastWriteTime,name
```

# 环境变量

我们如何执行外部命令，有两个条件:

+ `&"program"`，即将可执行文件用双引号括起来，前面再加一个`&`即可
+ 将环境添加到系统路径中
